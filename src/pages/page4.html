Autres logiciels utiles

Scilab

Scilab est un logiciel de programmation open-source qui permet de réaliser des calculs numériques. Il est utile pour faire du calcul matriciel, de la simulation, de l'optimisation et de la visualisation de données.
Dans certaines Universités, sa pratique est incluse dans certaines UE.
Vous pouvez le télécharger depuis le site officiel : Télécharger Scilab.

Il n'y a pas de cours Open Classroom dédié, mais ce document en libre service directement depuis le site : Scilab pour débutants

Matlab et Octave

Matlab est similaire à scilab ci dessus, il permet aussi de faire des calculs numériques, de simuler etc. et son point fort semble être ses outils graphiques et ses outils de simulation.
Son utilisation est donc courante dans les domaines "périphériques" aux Mathématiques comme la plupart des sous-domaines de la Physique…
Le problème est que ce logiciel est commercial, les Universités doivent disposer de licences permettant aux étudiants de l'installer en local… ou alors l'Université met à disposition une machine virtuelle (ou des machines physiques) sur laquelle fonctionne Matlab…
Pour s'y faire, je vous invite à vous rapprocher du service informatique de votre Université dans le cas où vous le voudriez en local.

Il y a une alternative open-source à Matlab : il s'appelle Octave et permet d'exéciter dez scripts Matlab.
Vous pouvez le télécharger ici : Télécharger Octave

Bien évidemment, des différences sont à craindre entre Matlab et Octave et ce malgré les contributions du projet Octave…

Même problème que pour Scilab, et visiblement, MathWorks (qui a conçu Matlab) propose lui aussi son propre cours en ligne : Tutoriels Matlab

C-C++

Le langage C est un langage de programmation largement utilisé qui offre (ou impose) au développeur un contrôle sur la gestion de mémoire qu'il utilise (contrairement à Python, "plus permissif" d'une certaine manière).
Par ailleurs, le langage C++ est une sorte de C amélioré qui permet d'implémenter plus de choses et qui offre plus "d'options de programmaton" en quelque sorte.
Il arrive que certaines Universités imposent son utilisation dans une Licence de Mathématiques

Si ça vous semble un peu opaque,vous pouvez vous rendre sur les pages Wikipédia correspondantes de C et de C++.
Et dans le cas où la pratique du C est dans votre cursus, votre enseignant abordera probablement les spécificités des deux langages liés par leur histoire.
Pour la faire courte : C est très vieux donc paraît plus "brut" et plus "exigeant" envers celui qui l'utilise, et C++ a historiquement été construit à partir de C pour permettre aux développeurs d'avoir un plus large éventail de possibilités et d'utilisation du langage.

Je vous conseille ces cours, dans lesquels ils expliquent comment configurer un IDE pour respectivement C et C++ sur votre PC :
Pour apprendre les bases de C : Apprendre à programmer en C
Pour apprendre les bases de C++ : Apprendre à programmer en C++
Pour apprendre le C++ en orienté objet : Programmer en orienté objet avec C++

La programmation orienté objet est une façon de programmer qui invite l'utilisateur à définir ses propores objets sur lesquels il va travailler…
cette façon de programmer offre une énorme quantité de possibilités, et est donc extrêmement utile pour simuler à peu près tous les phénomènes physiques imaginables.

SQL

"Structured Query Language" ou SQL est un langage informatique servant à exploiter les bases de données, et les bases de données relaionnelles.
Il est incontournable pour la gestion de bases de données.

Je vous invite à suivre ce cours : Implementez vos bases de données SQL.
