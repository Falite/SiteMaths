Installer et débuter sur Jupyter

En licence de mathématiques, il est souvent demandé de maîtriser certains langages de programmation comme Python, Sagemaths, Matlab, C/C++… (dépendemment du choix des Universités). En gros il est inévitable de passer par la programmation d'algorithmes mathématiques, de calculs numériques et de visualisations de données. Pour cela, Jupyter est un outil très utile et largement utilisé dans le milieu académique et professionnel. Il permet de créer des notebooks interactifs où l'on peut écrire du code, exécuter des cellules de code, visualiser des graphiques et ajouter du texte explicatif. Et quels langages ? Jupyter prend en charge de nombreux langages de programmation, notamment Python, R, Sage, C, Matlab (si votre Université a la licence chez Mathworks), et bien d'autres. Jupyter fonctionne en local, c'est à dire que vous l'installez sur votre ordinateur et il se lance sur votre navigateur par défaut.

Jupyter est un environnement de développement interactif qui permet de créer des notebooks contenant du code, des visualisations et du texte. Pour l'installer, suivez les étapes ci-dessous :

  1. Assurez-vous d'avoir Python installé sur votre machine. Vous pouvez télécharger Python depuis le site officiel : Télécharger Python.
  2. Ouvrez un terminal ou une invite de commande.
  3. Vérifiez que Python et son gestionnaire de paquets pip est installé en tapant la commande suivante : 
    python --version
    pip --version
  4. Installez Jupyter en utilisant pip, le gestionnaire de paquets de Python, avec la commande suivante : 
    pip install jupyterlab
  5. Une fois l'installation terminée, vous pouvez lancer Jupyter Notebook en tapant la commande suivante dans le terminal : 
    jupyterlab

    6. À ce stade, vous pouvez coder en Python sur JupyterLab

Sage est un langage différent de Python, pour l'utiliser sur JupyterLab, vous devez installer Sage puis installer son noyau dans Jupyter.
Ci dessous un bouton pour accéder à SageCell, un site sur lequel vous pouvez d'ores et déjà utiliser Sage sans installation.

  1. Téléchargez l'application Sage depuis le site officiel : Télécharger Sage.
  2. Installez Sage en suivant les instructions fournies sur le site.
  3. Une fois Sage installé, ouvrez un terminal et naviguez jusqu'au répertoire d'installation de Sage.
  4. Installez le noyau Sage pour Jupyter avec la commande suivante : 
    ./sage -pip install sagemath-jupyter
  5. Ensuite, lancez JupyterLab comme précédemment : 
    jupyterlab
